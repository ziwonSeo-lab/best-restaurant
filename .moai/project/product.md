# Best Restaurant Map (한국 맛집 지도)

## 프로젝트 개요

한국 전역의 인증된 맛집을 인터랙티브 지도에서 탐색할 수 있는 웹 애플리케이션입니다.

## 대상 사용자

- 맛집을 찾는 일반 사용자
- 여행 중 근처 인증 맛집을 찾고 싶은 여행객
- 모범식당, 미쉐린 인증 레스토랑에 관심 있는 미식가

## 핵심 기능

### 1. 3종 인증 맛집 표시
- **모범식당**: 정부 인증 모범 음식점
- **블루리본**: 미쉐린 블루리본 인정 레스토랑
- **빕 구르망**: 미쉐린 가성비 추천 레스토랑

### 2. 지역 선택
- 17개 시도 지역 선택 (서울, 부산, 대구, 인천, 광주, 대전, 울산, 세종, 경기, 강원, 충북, 충남, 전북, 전남, 경북, 경남, 제주)

### 3. 다중 필터링
- 음식 종류 필터 (한식, 일식, 중식 등)
- 소스 필터 (모범식당/블루리본/빕 구르망)
- 텍스트 검색 (식당명, 주소, 음식 종류)
- GPS 위치 기반 반경 필터 (500m ~ 5km)

### 4. 편의 기능
- 즐겨찾기 (localStorage 기반 북마크)
- 랜덤 맛집 추천
- 네이버 지도 연동 (상세 위치 검색)

### 5. 모바일 지원
- PWA (홈 화면 추가) 지원
- 반응형 모바일 퍼스트 디자인

## 사용 사례

1. **근처 맛집 찾기**: GPS 위치를 활성화하고 반경 필터로 주변 인증 맛집 탐색
2. **여행 계획**: 특정 지역의 모범식당/미쉐린 레스토랑 사전 조사
3. **음식 종류별 탐색**: 한식, 일식, 중식 등 원하는 음식 종류로 필터링
4. **즐겨찾기 관리**: 관심 있는 식당을 북마크하여 나중에 확인
