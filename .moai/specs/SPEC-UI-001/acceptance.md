# SPEC-UI-001 인수 기준

## 시나리오 1: 오래된 모범식당 경고 표시

**Given** 지정일이 "19920728"인 모범식당이 존재할 때
**When** 사용자가 해당 식당의 RestaurantCard를 볼 때
**Then** "1992년 7월" 형식으로 지정일이 표시되고
**And** "⚠️ 상호 변경 가능 - 최신 정보는 네이버지도에서 확인하세요" 경고 문구가 표시되어야 한다

## 시나리오 2: 최근 지정 모범식당 (경고 없음)

**Given** 지정일이 현재로부터 3년 이내인 모범식당이 존재할 때
**When** 사용자가 해당 식당의 RestaurantCard를 볼 때
**Then** 지정일이 포맷팅되어 표시되고
**And** 경고 문구는 표시되지 않아야 한다

## 시나리오 3: 최근 지정 필터 동작

**Given** 소스 필터가 "모범식당"으로 선택된 상태에서
**When** 사용자가 "최근 지정" 필터를 활성화할 때
**Then** 지정일이 5년 이내인 모범식당만 지도에 표시되어야 한다
**And** 5년 이상 된 모범식당은 지도에서 제외되어야 한다

## 시나리오 4: 비모범식당에는 미적용

**Given** 블루리본 또는 빕 구르망 식당이 존재할 때
**When** 사용자가 해당 식당의 RestaurantCard를 볼 때
**Then** 지정일 경고 관련 UI가 표시되지 않아야 한다

## 시나리오 5: 지정일 데이터 없음

**Given** designatedDate가 빈 문자열인 모범식당이 존재할 때
**When** 사용자가 해당 식당의 RestaurantCard를 볼 때
**Then** "지정일 정보 없음"으로 표시되어야 한다
**And** 경고 문구는 표시되지 않아야 한다

## 시나리오 6: 네이버지도 링크 동작

**Given** 5년 이상 된 모범식당의 경고 문구가 표시된 상태에서
**When** 사용자가 "네이버지도에서 확인하세요" 링크를 클릭할 때
**Then** 네이버지도에서 해당 식당명+주소 검색 결과가 새 탭에서 열려야 한다

## 품질 기준

- [ ] 날짜 포맷팅이 모든 모범식당에서 올바르게 동작
- [ ] 경고 문구가 모바일/데스크톱에서 적절히 표시
- [ ] 필터 토글이 기존 필터와 일관된 UX 유지
- [ ] 블루리본/빕 구르망 식당에 영향 없음
- [ ] Static Export 빌드 정상 동작
